Terminal Basics

ls - list what is in directory
cd - change directory to…
cd .. - move up in directory
python3 directory/python-code - running code
pip install --upgrade - to upgrade any library
mkdir - make a directory (folder)
Control C - quit in the terminal
‘’’ - triple quote to comment more than one line
Cmd / - comments out more than one line of code using #
