City of Ottawa Red-light Data Analysis
Project Breakdown →

Finally, I had gotten to a place in my confidence in Python coding that I wanted to venture down the path for data analysis. The City of Ottawa has open source data and I found (and used) their data for red light violations from 2016-2020. Here I analyzed questions centered around ROI for the City of Ottawa: which months/locations have the highest/lowest number of violations, and was there a statistically significant change in number of violations for the year 2020 in comparison to 2019. I used Pandas to analyze the data, matplotlib and NumPy to visualize the data, Folium to create map visualizations, and SciPy to run t-tests to analyze correlations.


Costco Deal Hunter
Project Breakdown →

After the first web scraping two projects - with help from articles I found, I decided to venture out on my own. I decided to make a web scraper that pulls the information for Costco’s coupon section and looks for keywords. I had to learn to use Selenium because it required user input for the location before allowing the coupons to load (as they are based on the location). I also used Pandas again to make the data frame that stores all the coupon information and then also a second data frame that stores only the relevant coupons for the keyword I inputted. I wanted to automatically run it once a week using Cron, however, I ran into some issues with Cron on a Macbook. I was anxious to get to my next project of data analysis so I moved on but there are so many ways this project can be taken further (which I hope to do): update my code once I fix the Cron/Macbook error, include a list of keywords to look for (as opposed to just one), making an email alert with the items listed based on the keywords inputted, and maybe even asking the user to input what words they want to set alerts for.
