Costco Deal Hunter
Project Code →

After the first web scraping two projects - with help from articles I found, I decided to venture out on my own. I decided to make a web scraper that pulls the information for Costco’s coupon section and looks for keywords. I had to learn to use Selenium because it required user input for the location before allowing the coupons to load (as they are based on the location). I also used Pandas again to make the data frame that stores all the coupon information and then also a second data frame that stores only the relevant coupons for the keyword I inputted. I wanted to automatically run it once a week using Cron, however, I ran into some issues with Cron on a Macbook. So I plan to update my code once I fix the Cron/Macbook error. I also plan to include a list of keywords to look for, as opposed to just one. This project can be taken further by making an email alert with the items listed based on the keywords inputted. It can be taken even further by asking the user to input what words they want to set alerts for.
