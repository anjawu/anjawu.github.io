My Website

Building My Own Website

The very first thing I did was I decided what material I wanted on my website and about how I wanted it to look. Then I drew on a piece of paper a rough idea of what I wanted, along with my must haves for the template. I knew I wanted three tabs but two of those three tabs I wanted a drop down menu so people can select from them. From there, I looked at templates on free-css - other websites are okay too, I just found one I liked on here. The one I picked was based on having what I needed (drop down menus) and also had colours and text style I liked (you can always change it but based on the scope of this project, I wanted to simplify my life).

Once you have downloaded the template, there will be a lot of folders. You want to focus on the files that have .html at the end. If you try to open by double clicking the file, it will mostly likely open the webpage in a browser. If you want to edit the code, you can right-click and click open with "your text editor/IDE".

**Please note that when I type out the code I will be putting a space beside "< div" for the starting container and a space after "< /div" for the ending container. I will be doing this because otherwise the code will not display but rather turn into html that is visually displayed.

3: Home Page

How my home page looks like at the time of writing this:

For the home page the layout I wanted was slightly different than the index2.html that was the template, but had the same basic idea.

I did not wanted to animated columns under the subheader, so I deleted the whole section labeled:

================================================== -->

Then after the animated columns for the header, I wanted to have a paragraph text underneath. So I created a paragraph, and I selected the class="twelve columns”. This is because I looked at the HTML code for columns.html and saw which column style I wanted. In this case I wanted to have one column, which coincided with class=”twelve columns”. So my code looked like:

< p>
< /p>
< /div>

When typing paragraphs, if you want a break in the line but want to continue in the same paragraph section, you can type:

< br>

Next, I did not want three columns but rather two, so once again using the same method as before I found the class I wanted was class= "six columns". And within these columns I did not want a button that says learn more, but rather I wanted a list that has the various pages on my webpage with their links. So in order to find what types of code this specific template had for lists, I went to the topology.html code. The label of what I wanted was < ul class="disc">. So my code looked like this for both columns:

< div class="six columns">
< h5>Coding Information< /h5>
< ul class="disc">
< li>< a href="github.html">GitHub< /a> - description of page.< /li>
< li>< a href=".html">< /a>< /li>
< li>< a href=".html">< /a>< /li>
< /ul>
< /div>

I didn't need a section on testimonials or clients, so I changed it to an "About Me" section. I looked through the template pages to see if there was a style I liked and I found on the feature -> elements page there was a panel that I liked. So I deleted the carousel of clients, added the panel (taking up two thirds of the row) and added a picture to the right of the text (a third of the row).

  • To make the panel two-thirds of the row, I put the class to be "eight columns".
  • To make the picture one-third of the row, I put the class to be "four columns".
  • To add an image you just need this code starting point:
    < img src="" width="" height=”auto” alt="">
    → The folder in which the picture is located goes in the "" after src. Be sure to keep the images in one folder (preferably called images) and contain that folder within your website folder.
    → You could do width by %, I didn’t want my image to change size according to the page size, so I always used 400 as my starting point and then adjusted based on how the image looked (because I've always been terrible at estimating size, I'd have to do trial and error).
    → In order to get the image center aligned just add < center>< /center> around your image code:
    < center>< img src="" width="" height=”auto” alt="">< /center>
