Projects

  • Red Light 🚦 (Python)
    When I was driving into downtown for work during the later part of COVID I noticed that the traffic still seemed to be a lot lighter than before. So I was curious what has changed on the roads. When I looked on the City of Ottawa open source data, I got very excited to see that they have data from red light violations from 2016-2020. So I used the open source red light camera violations from the City of Ottawa website to do an in-depth analysis of when and where the highest number of violations occur in order to give suggestions on next steps for the City of Ottawa planning new redlight traffic lights. There was the 2021 dataset added after I finished the report, so I went back and got new insights from 2016-2021.
  • Costco 🛒 (Python)
    I love a good Costco deal and now with two kids under two - one who sometimes eats the same amount as me - I figured it would be a good idea to make a Costco sale tracker for specific items I may want. So I made a web scraper for the coupons section of Costco, that pulls the coupon information and searches for keywords, using my previous learnings of web scraping combined with Selenium. The second part of this project involved scraping Real Canadian Superstore for the same items that were scraped from Costco and pulling prices. Then there was a price comparison done between Costco and Real Canadian Superstore to see which store is cheaper and by how much.
  • School boards 🏫 (SQL)
    As a teacher it is impossible to not pull trends occurring in your class and extrapolate it, even though it is a small sample size. Oftentimes we don’t use data to check if our feelings are correct but I found an open source dataset from the Ontario government that looks at the school boards across Ontario and their reading EQAO/OSSLT for 2018/2019, four- and five-year graduation rates and compares progress in each category since 2017/2018. Interestingly, they do not include the math results (missing both grade 6 and 9). So I jumped on the opportunity to try to answer some questions I had. I used BigQuery to query the data and Looker Studio to create a dashboard.
  • Website 🖥️ (HTML/CSS)
    I created this website using a template and then modified the pages to have the layout I wanted. I also included Google Analystics tracking and modified the code to be able to find out timeframes of how long someone spends on each page.
  • Indeed 💼 (Python)
    I used a guideline that scraped the Monster job website and made my own web scraper for specific job search on Indeed.ca. This was my second project using Python. It uses BeautifulSoup and requests libraries.
  • IMDb 🎥 (Python)
    I used a guideline that walked me through scraping IMDb top 100 movies and creating a data frame along with saving it to a CSV file. It uses BeautifulSoup, requests, and Pandas libraries.
