Website Details

During my coding (and data analysis) journey I had begun documenting how I was learning stuff as I was doing projects and research. I wanted to learn how to not only analyze data but also get it - so most of the projects I have done thus far have been about web scraping and my latest finally gets into a deep data analysis. So initially this website was primarily about using Python, but I started using SQL and have really enjoyed it thus far. This webpage contains what I have learnt - written in a way that was supposed to help me learn and reflect on my learning - so I often go on tangents about errors I had to try to solve for and research I had to do to complete a project. I also link to a lot of good resources from the people I have learnt from. And I include my final code linked via GitHub.

How to navigate this website you ask? The pages contain a drop down menu to try to further organize my thoughts, plus the Projects page has a summary of each project with technologies used and skills learnt. Breakdown of what you will find:

Coding Information
  • SQL - has syntax (beginner to advanced) I have learnt using SQL along with Mode reports for sample projects.
  • Python Libraries - Python has a LOT of libraries, so what I have done is write about the ones I have used extensively so far.
  • GitHub - is such a good way to store your code and share it, but it seemed overwhelming to me when I first started, so I wrote down stuff I had to learn while doing it.
  • Google Analytics - was created for this website with an alteration from the standard Google Analytics setup to account for time spent on a page. I began exploring the functionality of Google Analytics and then I decided to learn Google Data Studios to allow me to analyze the data and create reports.
  • Fun Things Learnt - learning is always so much fun and there is always so much new information, it can be hard to not travel down rabbit holes while researching. So I decided to make this section for two main reasons: 1) I wanted to write down new things that I had learnt that I haven’t had a chance to try out in one of my big projects and 2) By having to type out what I learnt it prevented me from going down too many rabbit holes and really zeroing in on what I was trying to figure out.
  • Versus - Like with learning new languages, there are plenty of words that seem similar but are different and you mix up for whatever reason, this is what this section is for me. I look at terms that seem to be used interchangeably that I just couldn’t keep straight.
  • Environment Setup - setting up your code editor and terminal is all about choosing your own style and making sure you have it set up in a way that is conducive to your programming habits. In this section, I go over how I set mine up to give a starting point.
Projects
  • Red-light - When I was driving into downtown for work during the later part of COVID I noticed that the traffic still seemed to be a lot lighter than before. So I was curious what has changed on the roads. When I looked on the City of Ottawa open source data, I got very excited to see that they have data from red light violations from 2016-2020. So I used the open source red light camera violations from the City of Ottawa website to do an in-depth analysis of when and where the highest number of violations occur in order to give suggestions on next steps for the City of Ottawa planning new redlight traffic lights.
  • Costco - I love a good Costco deal and now with two kids under two - one who sometimes eats the same amount as me - I figured it would be a good idea to make a Costco sale tracker for specific items I may want. So I made a web scraper for the coupons section of Costco, that pulls the coupon information and searches for keywords, using my previous learnings of web scraping combined with Selenium.
  • Website - I created this website using a template and then modified the pages to have the layout I wanted.
  • Indeed - I used a guideline that scraped the Monster job website and made my own web scraper for specific job search on Indeed.ca. This was my second project using Python. It uses BeautifulSoup and requests libraries.
  • IMDb - I used a guideline that walked me through scraping IMDb top 100 movies and creating a data frame along with saving it to a CSV file. It uses BeautifulSoup, requests, and Pandas libraries.
  • Hangman - my very first Python project was making a hangman game.
